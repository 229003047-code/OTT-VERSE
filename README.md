# 🎬 OTT Verse

A Netflix-inspired entertainment streaming platform UI, built to explore modern web design patterns for content discovery — hero banners, trending carousels, and curated category rows.

**🔗 Live Demo:** [ott-verse.vercel.app](https://ott-verse.vercel.app/)



## ✨ Features

- **Hero Banner** — auto-rotating featured title with Play / More Info actions
- **Top 10 Trending** — ranked carousel with numbered badges
- **Category Rows** — Top Rated, Prime Video, Hotstar, Sci-Fi Movies, and more
- **Search Bar** — quick movie lookup from the navigation bar
- **My List** — navigation entry for saved/favorited content
- **Responsive Carousels** — horizontal scroll with left/right navigation arrows
- **Dark Theme UI** — high-contrast, poster-first design matching modern OTT platforms

---

## 🛠️ Tech Stack

- **React** — component-based UI
- **Vite** — fast dev server and build tooling
- **CSS** — custom styling for layout and theming
- **ESLint** — code quality and consistency
- **Vercel** — deployment and hosting

## 🚀 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/229003047-code/OTT-VERSE.git
cd OTT-VERSE
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
